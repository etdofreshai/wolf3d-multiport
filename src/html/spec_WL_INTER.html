WL_INTER.C — Intermission Screens Game

Purpose: Between-level intermission screens: level completion tallies (kills, secrets, treasure, time, par), high score display and entry, the PG-13 warning screen, the victory sequence, episode end screens, preload progress bar, and copy protection.

Functions

  • void LevelCompleted(void) — Show level completion tally screen with animated counting
  • void Victory(void) — Episode victory sequence
  • void PG13(void) — Display PG-13 warning screen
  • void DrawHighScores(void) — Render the high score table
  • void CheckHighScore(long score, word other) — Check if score qualifies, handle name entry
  • void PreloadGraphics(void) — Preload graphics with progress bar
  • boolean PreloadUpdate(unsigned current, unsigned total) — Update preload progress display
  • void ClearSplitVWB(void) — Clear split-screen VWB state
  • void Write(int x, int y, char *string) — Write text on intermission screens
  • void BJ_Breathe(void) — Animate BJ breathing during tally
  • void EndScreen(int palette, int screen) — Display an end-of-episode screen
  • void EndSpear(void) — Spear of Destiny ending sequence
  • void CopyProtection(void) — Copy protection check (manual lookup)

Dependencies

  • WL_DEF.H — Game definitions
  • ID_VH.C — Drawing routines
  • ID_IN.C — Input for acknowledgement
  • ID_SD.C — Music and sound effects
  • ID_CA.C — Cache graphics for display

Porting Notes

C SDL3
The tally screen uses blocking animation loops with VBL waits. These translate to SDL_Delay-based loops. The BJ breathing animation is a simple frame alternation timer. Copy protection can be removed.
TypeScript
The animated counting sequences need to be converted from blocking loops to incremental per-frame updates. Each counter (kills, secrets, treasure, time) counts up independently.
