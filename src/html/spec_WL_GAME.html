WL_GAME.C — Game Loop Game

Purpose: Contains the outer game loop (GameLoop), level setup (SetupGameLevel), map scanning (ScanInfoPlane), play screen drawing, demo recording/playback control, death sequence, and the level-to-level progression logic. This is the bridge between the main menu system and the actual gameplay in WL_PLAY.

Key Globals

Functions

  • void GameLoop(void) — Outer game loop: load level, play, check win/death/continue
  • void SetupGameLevel(void) — Load map, initialize tile arrays, spawn actors and items
  • void ScanInfoPlane(void) — Parse map info plane: spawn enemies, items, doors from tile values
  • void DrawPlayScreen(void) — Draw the HUD status bar, weapon, face, score, etc.
  • void DrawPlayBorder(void) — Draw border around viewport
  • void Died(void) — Death sequence: spin camera, fade to red
  • void ClearMemory(void) — Purge all cached assets for level transition
  • void PlaySoundLocGlobal(word s, fixed gx, fixed gy) — Play positioned sound effect
  • void UpdateSoundLoc(void) — Update stereo panning based on player position
  • void StartDemoRecord(int levelnumber) — Begin recording a demo
  • void FinishDemoRecord(void) — Finalize and save recorded demo
  • void RecordDemo(void) — Interactive demo recording
  • void PlayDemo(int demonumber) — Play back a recorded demo

Dependencies

  • WL_DEF.H — All game definitions
  • WL_PLAY.C — PlayLoop (inner game loop)
  • WL_ACT1.C, WL_ACT2.C — Actor/door spawning
  • WL_AGENT.C — Player spawning, status bar drawing
  • ID_CA.C — Map loading, asset caching
  • ID_VH.C — LoadLatchMem, FizzleFade

Porting Notes

C SDL3
Game logic is mostly portable. The GameLoop blocking-while structure works in SDL since we control the main loop. Level setup and map scanning are pure data manipulation. Sound positioning uses the same left/right angle calculation.
TypeScript
GameLoop must be restructured as a state machine since it cannot block. The death sequence (Died) with its spinning camera needs to yield frames. ScanInfoPlane is pure data parsing — straightforward translation.
